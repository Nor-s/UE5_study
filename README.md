# CH5 폰의 제작과 조작

- 폰 = 움직이는 액터 + 조종당하는 기능
- 폰을 직접 조작 (입력 설정)
  - 폰 무브먼트 컴포넌트
  - 애니메이션 블프 사용

## 폰의 구성요소

- 시각적 요소
  - 인간형 폰: 애니메이션 기능 필요
  - 스케레탈 메시: 리깅 데이터 추가됨 (관리하는 컴포넌트: 스켈레탈 메시 컴포넌트)

- 충돌요소
  - 인간형은 보통 캡츌 컴포넌트 사용
- 움직임 요소
  - 폰 무브먼트 컴포넌트 
  - FloatingPawnMovement, CharacterMovement
- 내비게이션
  - 폰은 언리얼 엔진의 내비게이션 시스템과 연동됨, 길찾기 기능 가짐

- 카메라 출력
  - 이동과 동시에 모니터에 장면을 출력해야함
  - 폰에 카메라 컴포넌트를 부착하면, 플레이어 컨트롤러가 폰에 빙의할 때 자동으로 폰에 부착된 카메라를 모니터에 출력
